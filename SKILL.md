# MiniMax TTS Skill

专业的 TTS 语音合成 skill，专为有声书、儿童绘本、视频旁白设计。

## ✨ 特性

- 🎯 **智能推荐**：根据内容自动推荐最合适的音色
- 🎵 **样音缓存**：本地缓存样音，避免重复生成
- 📖 **语义分析**：基于有声书演播规则，智能添加停顿
- 💬 **讲述感优化**：主谓分离、时间转换、并列结构自动处理
- 🎭 **情感语速**：辛苦场景×0.92，欢快场景×1.05
- 📦 **批量生成**：支持多场景批量生成音频
- 💰 **费用透明**：实时估算费用

## 🚀 快速开始

### 1. 配置 API Key

在 `.env` 中配置：

```bash
MINIMAX_API_KEY=sk-api-xxx
MINIMAX_BASE_URL=https://api.minimax.io
DEFAULT_MODEL=speech-2.6-turbo
```

### 2. 推荐音色

```bash
python scripts/voice_selector.py "你的旁白内容"
```

输出：
- 内容分析（类型、受众、情感）
- 推荐音色列表（带样音路径）
- 推荐理由

### 3. 批量生成音频

```bash
python scripts/batch_generator.py \
  --voice "Chinese (Mandarin)_Warm_Girl" \
  --input scenes.yaml \
  --output ./outputs/my_story
```

## 📖 最佳实践

### 停顿规则（基于有声书演播）

| 标点/语义 | 推荐停顿 | 说明 |
|----------|---------|------|
| 逗号 | 0.15秒 | 自然呼吸 |
| 句号 | 0.25秒 | 句子结束 |
| 问号 | 0.35秒 | 问句停顿 |
| 感叹号 | 0.30秒 | 情感表达 |
| 顿号 | 0.12秒 | 并列成分 |
| 语义停顿 | 0.1-0.2秒 | 主谓分离、时间转换 |
| 开头引起注意 | 0.2秒 | "看"、"瞧"等 |

### 情感语速调整

| 情感 | 语速倍数 | 示例场景 |
|------|---------|---------|
| 辛苦 | ×0.92 | "烈日当头，汗水滴进了泥土里" |
| 感动 | ×0.95 | "每一粒米饭都是礼物" |
| 中性 | ×1.0 | 一般叙述 |
| 欢快 | ×1.05 | "今天我们是光盘小明星！" |
| 疑问 | ×1.0 | "你知道小米粒怎么长大的吗？" |

### 语义停顿标记

在文本中使用 `<#秒数#>` 标记停顿：

```yaml
text: "看<#0.2#><#0.15#>你的小碗里盛着冒热气的白米饭<#0.25#>"
```

**关键语义停顿点：**

1. **开头引起注意**："看"、"瞧"、"听" 后停顿 0.2秒
2. **时间转换**："早春时节"、"过了很久" 前停顿 0.15-0.2秒
3. **主谓分离**："农民伯伯<#0.1#>弯下腰" 中间停顿 0.1秒
4. **并列结构**："拔草<#0.12#>浇水" 中间停顿 0.12秒
5. **强调词前**："都是"、"就是" 前停顿 0.15秒

### 重音词识别

以下类型的词需要重读：

- **形容词**：冒热气、沉甸甸、亮晶晶、光溜溜
- **强调词**：每一粒、全部、最好、真、非常
- **动作动词**：弯下腰、种下、滴进、吃光
- **情感词**：感谢、礼物、辛苦、香、甜

## 🔧 工作流程

```
用户请求
    ↓
分析内容特征（类型、情感、受众）
    ↓
推荐音色 → 检查本地样音
    ↓           ↓
 有样音     无样音
    ↓           ↓
直接返回   生成并缓存
    ↓           ↓
用户试听 ← ← ← ←
    ↓
选择音色
    ↓
语义分析（停顿、重音、语速）
    ↓
批量生成
    ↓
返回音频路径
```
